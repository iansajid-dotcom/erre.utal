Lectura de datos, dplyr y primeras graficas con ggplot2

Trabajamos el mismo dataset (d1.nativas) en sus tres formatos para practicar formas de importar datos a R: read.table(), read.csv() y read_excel(). Despues dplyr (select, filter, arrange, mutate, group_by + summarise, joins, el operador pipe) y cerramos con las primeras capas de ggplot2. 

nat.t <- read.table("d1.nativas.txt", header = TRUE)
nat.c <- read.csv("d1.nativas.csv", header = TRUE,
                   stringsAsFactors = TRUE,
                   na.strings = c("NAN","Na","na","NA"))
library(readxl)
nat.x <- read_excel("d1.nativas.xlsx", sheet = "bosque", na = "na")
