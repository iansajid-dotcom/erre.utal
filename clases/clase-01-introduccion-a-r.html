Introduccion a R y RStudio

Se dicto en dos sesiones: fundamentos del lenguaje orientado a objetos, tipos de datos basicos (numeric, character, logical), vectores y sus operaciones (indexacion, orden, pertenencia), matrices, y data frames con columnas de distinto tipo: creacion, exploracion, subsetting, ordenamiento y tablas de frecuencia.

Ejemplo de vectores e indexacion:

tor <- c(57, 7, 102, 2, 34, 11)  # vector

sort(tor)
tor[tor %in% c(3, 5, 11)]         # indexacion por pertenencia
tor[tor > 11]                     # indexacion logica

Ejemplo de data frame:

temps <- data.frame(
  tmin  = c(8.5, 11.5, 12.2, 14.3, 11.5, 9.1),
  tmax  = c(28.5, 27.1, 23.3, 16.6, 20.1, 21.7),
  dia   = c("lun","lun","mar","mar","mier","mier"),
  lugar = c("talca","chillan","talca","chillan","talca","chillan")
)

subset(temps, lugar == "chillan")
tapply(temps$tmin, temps$lugar, mean)
